# ПСК Омега — сайт

Лендинг для компании по укладке промышленных полов.

## Репозиторий

GitHub: `nickddd-hash/floor-website`
GitHub Pages: `https://nickddd-hash.github.io/floor-website/`

## Стек

- Чистый HTML / CSS / JS (без фреймворков)
- Один файл: `index.html`
- Стили: `css/style.css`
- Скрипты: `js/main.js`
- Картинки: `images/`

## Цвета и шрифты

- Акцентный цвет: `#85C226` (зелёный)
- Фон: тёмный (`#0d0d0d` / `#111`)
- Шрифт: Inter (Google Fonts)

## Структура страницы

1. **Header** — логотип Омега + навигация + кнопка «Связаться»
2. **Hero** — главный заголовок «Промышленные полы под ключ»
3. **Stats** — цифры (опыт, объекты, гарантия, срок)
4. **Services** — виды покрытий (карточки)
5. **Process** — этапы работы
6. **Partners** — логотипы партнёров
7. **CTA Banner** — призыв к действию
8. **Footer** — контакты, Telegram
